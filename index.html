Free Tools for Private Communication & File Sharing

Most online platforms are built to extract attention, data, and control. This page points to tools that prioritize privacy, autonomy, and resilience.

You don’t need all of these. Pick what fits your situation and your people.

Save file to use this page offline and share.

Messaging

  • Signal — Encrypted messaging app, widely used and practical.
    signal.org | source
  • Briar — Peer-to-peer messaging, works without internet (Android).
    briarproject.org | F-Droid
    Works over Bluetooth, Wi-Fi, or Tor.
  • Session — Anonymous messaging with no phone number.
    getsession.org | GitHub mirror
    Routed through a decentralized network.

File sharing

  • Syncthing — Decentralized file syncing between your devices.
    syncthing.net | GitHub mirror
    Works over internet or local networks; no cloud server.
  • OnionShare — Send files directly, no middleman.
    onionshare.org | GitHub mirror
    Uses Tor; hard to censor.
  • Magic Wormhole — One-time code encrypted file transfers.
    docs | GitHub mirror
    Good for sending files securely over the internet with minimal setup.
  • PairDrop — Browser-based local file sharing (no install).
    pairdrop.net | GitHub mirror
    AirDrop-like sharing in a web browser; great for quick one-off transfers.
  • IPFS — Content-addressed file sharing, ideal for mirrors.
    ipfs.tech | docs

Spreadsheets & documents

  • CryptPad — Encrypted collaborative docs and spreadsheets.
    cryptpad.fr/features.html | source
    Server cannot read your content; privacy-focused.
  • OnlyOffice — Full-feature office suite (docs, spreadsheets, presentations).
    onlyoffice.com
    Feature-rich, but not encrypted by default.
  • LibreOffice — Offline documents and spreadsheets, no account needed.
    libreoffice.org | source

Offline & local sharing

  • LocalSend — Share files over local Wi-Fi.
    localsend.org | GitHub mirror
    Fast and simple across Android, iOS, Windows, macOS, and Linux.
  • KDE Connect — Share files + clipboard + notifications between devices.
    kdeconnect.kde.org | source
    Great for ongoing “phone + laptop” workflows, especially on Linux/Android.
  • USB drives / MicroSD cards
    Still reliable. Works everywhere. Hard to block.
  • Local Wi-Fi servers
    One device shares files nearby without internet.

If you’re under surveillance, protest, or disaster conditions

  • Prefer offline tools: Briar (local mode), Syncthing over LAN, LocalSend, USB drives.
  • Use Tor Browser to access blocked websites safely.
  • Do not assume devices are secure; consider using a clean device for sensitive work.
  • Share files locally or via encrypted links instead of public cloud services.

About mirrors

Some tools are listed with more than one link. These are mirrors — alternate places to get the same software. If one site is blocked or disappears, the tool still exists elsewhere.

Tools don’t replace trust or community.
But the right tools can help protect them.
